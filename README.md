#Editor

Visual Studio Code: https://code.visualstudio.com/

#Usage

open default_plugins.txt

add "BotManager" plugin

	"plugin"
	{
		"name" "RCBot"
		"script" "BotManager/BotManager"
		"concommandns" "rcbot"
	}
	
when running type

as_command rcbot.addbot

to add a bot
